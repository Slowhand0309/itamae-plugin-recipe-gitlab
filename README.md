# Itamae::Plugin::Recipe::Gitlab



## Installation

Add this line to your application's Gemfile:

```ruby
gem 'itamae-plugin-recipe-gitlab'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install itamae-plugin-recipe-gitlab

## Usage

**gitlab::install**

* recipe

```ruby
include_recipe 'gitlab::install'
```
